┌─────────────────────────────────────────────────────────────────────┐
│ METHOD 1: USE PRE-BUILT PRESET (Mixamo → AccuRig)         45 seconds│
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  1. Select source armature (Mixamo)                                  │
│  2. Animation Manage → Save Animation → "MyAnim" → OK                │
│  3. Bone Mapping → Mapping Presets → Browse Mappings                 │
│     → examples/Mixamo_to_AccuRig.json → OK                           │
│  4. Select target armature (AccuRig)                                 │
│  5. Apply Animation → Apply Animation with Mapping                   │
│     → Select "MyAnim" → OK                                           │
│  6. Press SPACEBAR to play ✓                                         │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ METHOD 2: AUTO-MAP (Any Rig → Any Rig)                    60 seconds│
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  1. Select source → Animation Manage → Save Animation                │
│  2. Bone Mapping → Setup:                                            │
│     - Source Armature: [select]                                      │
│     - Target Armature: [select]                                      │
│     - Preset Name: "Source_to_Target"                                │
│     - Click: Create Bone Mapping                                     │
│  3. Auto-Map Threshold: 0.7                                          │
│     Click: Auto-Map Bones                                            │
│  4. Mapping Presets → Save Mapping → OK                              │
│  5. Select target armature                                           │
│  6. Apply Animation → Apply with Mapping → Select anim → OK          │
│  7. Press SPACEBAR to verify ✓                                       │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘

╔═══════════════════════════════════════════════════════════════════════╗
║                        THRESHOLD SETTINGS                             ║
╠═══════════════════════════════════════════════════════════════════════╣
║                                                                       ║
║  0.9  Very Strict    ▓▓▓▓▓▓▓▓▓░  60% mapped, 0% errors               ║
║  0.8  Strict         ▓▓▓▓▓▓▓▓░░  80% mapped, <1% errors              ║
║  0.7  Recommended ★  ▓▓▓▓▓▓▓░░░  95% mapped, <2% errors              ║
║  0.6  Permissive     ▓▓▓▓▓▓░░░░  100% mapped, 5% errors              ║
║  0.5  Too Low        ▓▓▓▓▓░░░░░  100% mapped, 10%+ errors            ║
║                                                                       ║
║  Use 0.7 for Mixamo → AccuRig                                         ║
║  Use 0.65-0.75 for custom rigs                                        ║
║                                                                       ║

┌─────────────────────────────────────────────────────────────────────┐
│ CONFIDENCE BADGES (In Bone Mappings List)                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  ✓        100% - Manual or exact match (perfect)                     │
│  90-99%   High confidence auto-map (very good)                       │

╔═══════════════════════════════════════════════════════════════════════╗
║                          QUICK CHECKS                                 ║
╠═══════════════════════════════════════════════════════════════════════╣
║                                                                       ║
║  BEFORE APPLYING:                                                     ║
║    □ Target armature is selected (orange highlight)                  ║
║    □ Bone mappings list is populated                                 ║
║    □ At least 90% of bones mapped                                    ║
║    □ No unmapped critical bones (Hips, Spine, Arms, Legs)            ║
║                                                                       ║
║  AFTER APPLYING:                                                      ║
║    □ Press SPACEBAR - animation plays                                ║
║    □ All body parts move naturally                                   ║
║    □ No console errors                                               ║
║    □ Left/right sides correct                                        ║
║                                                                       ║
╚═══════════════════════════════════════════════════════════════════════╝

┌─────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  PROBLEM: Animation doesn't apply                                    │
│  FIX: • Is target armature selected?                                 │
│       • Are there bone mappings in list?                             │
│       • Try clicking target armature again                           │
│                                                                       │
│  PROBLEM: Some body parts don't move                                 │
│  FIX: • Check console for unmapped bones                             │
│       • Lower threshold to 0.65 and auto-map again                   │
│       • Click "Validate Mapping" to see issues                       │
│                                                                       │
│  PROBLEM: Animation looks broken                                     │
│  FIX: • Review all mappings with confidence < 80%                    │
│       • Check left/right bones map to correct sides                  │
│       • Test with simple animation first (walk/idle)                 │
│                                                                       │
│  PROBLEM: Fingers don't animate                                      │
│  FIX: • Check if target rig has finger bones                         │
│       • Lower threshold to 0.6 for finger matching                   │
│       • Manually map finger bones if needed                          │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
